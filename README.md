# My Portfolio Site

モダンで洗練されたデザインのポートフォリオサイトです。
React と Vite を使用して構築されており、GitHub Pages で公開されています。

## 🚀 公開URL
**[https://nashi39.github.io/Portfolio/](https://nashi39.github.io/Portfolio/)**

## 🛠 使用技術
- **Frontend**: React, JavaScript
- **Styling**: Vanilla CSS (Modern UI Design)
- **Deployment**: GitHub Pages
- **Assistant**: Antigravity (Advanced Agentic Coding AI)

## 📁 プロジェクト内容
- **Universe War**: 2Dシューティングゲーム (C / DXLib)
- **Math Card**: 学習支援カードゲーム (C++ / DXLib)
- **Sosproject**: アニメ作品詳細Webサイト (React)
- **Shift Master**: シフト管理アプリケーション (React / Firebase)

## 開発環境のセットアップ
```bash
# 依存関係のインストール
npm install

# ローカルサーバー起動
npm run dev

# ビルドとデプロイ
npm run deploy
```
